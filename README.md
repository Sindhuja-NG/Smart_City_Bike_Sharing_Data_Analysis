# Smart_City_Bike_Sharing_Data_Analysis
Bike_Sharing_Data_Analysis

This project analyzes bike-sharing station data to understand bike availability, station distribution, operational status, capacity utilization, and demand patterns across different cities. The analysis was performed using Power BI with data cleaning, data modeling, DAX calculations, and interactive visualizations.

The dashboard provides insights into city-wise station performance, bike availability trends, station demand levels, and operational efficiency. Descriptive statistics such as average, median, and standard deviation were used to evaluate bike availability variations and identify opportunities for improving bike distribution.

Key outcomes include identifying major bike-sharing locations, analyzing station utilization, understanding demand patterns, and providing recommendations for better bike allocation and resource management.
